    Motif Graph

    Motif Graph is an online graph analysis platform built by Singapore Cylynx company based on Graphin graph visual analysis capabilities and AI graph intelligence technology. It supports risk monitoring to combat financial fraud and understands activities on the virtual asset chain through cross-chain analysis.

    Graph Studio

    Graph Studio is a one-stop graph development platform built by the Alibaba Cloud Dataworks Team based on Graph Compute. The graph analysis which is based on Graphin in this platform provides common graph analysis capabilities such as data retrieval, relationship diffusion, layout switching, node selection, association highlighting, and sharing, which can help users quickly complete related analysis work.
